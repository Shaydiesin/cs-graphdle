Find a Perfect Matching Cut

Color each vertex red or blue so that every vertex has exactly one neighbor of the opposite color.

Click a vertex to cycle through ⚪ → 🔴 → 🔵. When you're done, press Check.

Why "CS-Graphdle"? We wanted to call it Graphdle, but that name was already taken by a puzzle about plotting mathematical functions rather than graph-theoretic graphs. Computer scientists have been overloading the word graph for decades, so CS-Graphdle it is.

Reference

1) Neeldhara Misra, Yash More. Finding Perfect Matching Cuts Faster. Combinatorial Algorithms (IWOCA 2023), Springer, pp. 307–318.
